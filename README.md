# ProJuris Challenge

Backend challenge for ProJuris with Spring Boot

## Requirements
- Java version 11
- Maven 

## Quick start

1. Build locally with Maven

```bash
$ mvn clean package
```

2. Start
```bash
$ java -jar target/challenge-0.0.1-SNAPSHOT.jar
```

3. Check Swagger Docs 
http://localhost:8080/swagger-ui.html
